checkExists(pred, class_val, d_type) → {boolean}

Checks if a predicate is already in the parameter manager. This is done by checking the predicate, the class, and if there's a datatype already assigned. If class and predicate are true and the datatype is anything but nominal, the datatype will be overwritten (by returning true). Returns true if already exists, false if it does not.
